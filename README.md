# ViknaDOMI Telegram Bot

Це Telegram чат-бот, розроблений як інформаційний хаб (каталог) з функцією контрольованого доступу.

## Опис проекту

Бот реалізує функціонал закритого каталогу для співробітників або партнерів компанії. Основна ідея — надання доступу до структурованої інформації (документи, фото, відео, посилання на чати) лише авторизованим користувачам.

Референс: @EKIPAZH_ALUM_bot

## Ключові можливості

1.  **Система авторизації**:
    - Вхід у бот можливий лише після верифікації номера телефону.
    - Користувач відправляє контакт через вбудовану кнопку Telegram.
    - Адміністратор отримує сповіщення про нову заявку і може схвалити або відхилити її.
    - Доступ до меню бота відкривається тільки після схвалення.

2.  **Керування доступом**:
    - Адміністратор має право видаляти користувачів або блокувати їм доступ, якщо вони припиняють співпрацю з компанією.

3.  **Інформаційна структура (Планується)**:
    - Бот працюватиме як каталог із рубриками.
    - Основний контент: документи, зображення, відео, посилання на профільні чати та групи.

## Технічний стек

- **Мова**: Python 3.10+
- **Фреймворк**: aiogram 3.x
- **База даних**: SQLite (з використанням SQLAlchemy + aiosqlite)

## Налаштування та запуск

1.  Клонуйте репозиторій.
2.  Створіть файл `.env` на основі прикладу та вкажіть:
    ```env
    BOT_TOKEN=ваш_токен
    ADMIN_IDS=id_адміністратора1,id_адміністратора2
    ```
3.  Встановіть залежності:
    ```bash
    pip install -r requirements.txt
    ```
4.  Запустіть бота:
    ```bash
    python bot.py
    ```
